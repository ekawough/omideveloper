// faq

Plaud user questions, answered.

Does this break my Plaud Pro subscription?
No. Plaud Pro keeps working exactly as is. Your Plaud transcripts still appear in Plaud's app. The Omi connection is parallel, not a replacement. You can run both at the same time. Most users do.
Do I need to buy an Omi too?
No. The Omi mobile app is free and sufficient to run any Plaud Compatible app on this page. The Omi necklace adds real time capability (unlocks apps like Tesla Connect), but it is optional. The free Omi mobile app plus your existing Plaud is enough.
How does my Plaud audio reach the Omi app?
Plaud transfers recordings to its own cloud after capture. The xor connection (PR #3287 in BasedHardware/omi GitHub) tells the Omi mobile app to read those Plaud recordings as input audio, then run them through whatever Omi apps you have installed. No reset, no reflash, no firmware change on your Plaud.
What is the actual cost to run apps on Plaud?
Free for the Omi mobile app and connection. Each app on top has its own pricing. Daily Wrapped Pro is free. Coffee Close is $19/mo. Showing Notes is $29/mo with 7 day free trial. SalesSignal is $49/seat or $29 annual. No app charges anything extra for being routed through Plaud.
Will all future Omi apps work on Plaud?
Most will. New apps default to the Memory Trigger pattern, which is Plaud Compatible. The exceptions are real time apps that need live audio streaming (like Tesla Connect). Each app on this site is labeled with a Plaud Compatible badge or a "Omi necklace only" badge so you know up front.
What if I want to switch fully to Omi later?
Same apps, same accounts, same data. Buy the Omi necklace ($129 right now), pair it in the same Omi mobile app, and your installed apps switch to using the necklace's audio feed instead of Plaud's. Or run both at once. Whichever device is talking gets processed.
Where is the xor connection feature documented?
PR #3287 in github.com/BasedHardware/omi, merged into mobile release v1.0.75+450. Open the Omi mobile app, Settings, About, and confirm version 1.0.75+450 or later. If you are below that, update from the App Store or Play Store.
